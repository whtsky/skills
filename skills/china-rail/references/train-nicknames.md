# 中国铁路车型俗称速查

## 复兴号（CR）

CR400AF = 红神龙（银底红条涂装）
CR400BF = 金凤凰（车灯如凤眼，腰线似凤羽）
CR400BF-C = 瑞雪迎春（冬奥列车）
CR300AF = 蓝暖男（蓝色，250km/h）
CR300BF = 蓝妹妹（蓝色，250km/h）
CR200J 一代深绿 = 垃圾桶/立即桶/东福绿/绿巨人（深绿涂装像垃圾桶；陆东福任内推出）
CR200J 三代白色 = AD钙奶（配色像娃哈哈AD钙奶包装）

惨遭桶击 = 普速列车被 CR200J 替换
DF200J / CR145J = 西户铁路 DF4D 刷成 CR200J 涂装

## 和谐号（CRH）

CRH1 = 大地铁/耳膜快乐车（外观像地铁；不锈钢气密性差，过隧道耳朵疼）
CRH1A-A = 萌地铁
CRH1E = 睡地铁（动卧版）
CRH2 = 带鱼（车身细长）
CRH2C = 疯带鱼（高速版）
CRH2G/CRH2E二代 = 先进黑脸（黑色前脸）
CRH3/CRH380B = 兔子
CRH5 = 动驴
CRH380A = 老鼠（车头尖长）
CRH380C = 鲨鱼
CRH380D = 疯地铁（高速版大地铁）

烤兔子 = CRH380BL 京沪线定远站失火
5033 = CR400BF-5033 轴温预警大户（长客出品）

## 内燃机车（东风系列）

东风型两窗版 = 哭脸
DF4B 客运型 = 橘子（橙色）
DF4B 货运型 = 西瓜（绿色）
DF4B 货运型绿黄 = 武警（颜色像武警制服）
DF4D 提速客运 = 老虎
DF4D 准高速客运 = 花老虎
DF4D 货运 = 乌克兰（配色像倒挂乌克兰国旗）
DF8B = 芭比
DF11 = 狮子/妖妖
DF11G = 猪头（交流=小猪，直流=大猪，拆分=拆猪）
DF11Z = 大水表
ND5 = 大老美（美国造）
HXN5 = 大蚂蚱
HXN5B = 番茄炒鸡蛋
NDJ3 = 大白猪

## 电力机车（韶山/和谐系列）

SS1 = 芍药
SS7C = 大脸猫/香蕉
SS7D = 金刚
SS7E = 美女/火腿肠
SS8 = 扫把
SS9 = 青蛙
SS9G = 烧酒
HXD1D = 红枣
HXD2 = 河马
HXD3 = 电猴

HXD3D = 番茄

## 涂装

原色 = 刷绿前标准涂装（红皮车、蓝皮车、白皮车）
伪绿皮/高阻绿/阻母绿 = 刷绿后统一绿色涂装
绿皮车 = 泛指所有普速列车
